The MoodMatrix Recommendation System is an AI based web application designed to provide personalized movie suggestions according to the user’s current mood and emotional state. With the rapid growth of online streaming platforms, users often face difficulty selecting suitable content from a vast number of available movies. This project aims to solve that problem by integrating mood selection and sentiment analysis to improve recommendation accuracy. The system collects user input in the form of direct mood selection and text description of feelings. Using Natural Language Processing techniques, the system analyzes sentiment polarity and maps it to predefined mood categories. A hybrid filtering approach is then applied by combining mood, genre, and movie rating from the dataset to generate relevant suggestions. The application is developed using Python and Flask, with data processing and visualization support, ensuring a practical and intelligent recommendation experience.
